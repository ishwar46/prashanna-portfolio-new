Add a JSON-LD script tag to page.tsx following the official Next.js pattern (https://nextjs.org/docs/app/guides/json-ld).

2. Define `BASE_URL` from `process.env.NEXT_PUBLIC_SITE_URL || "https://prashannasangroula.com"`
3. Build the JSON-LD object with `@context: "https://schema.org"` and `@graph` array containing three schema types:

**Person schema:**
- @type: "Person", @id: `${BASE_URL}/#person`
- name: "Prashanna Sangroula"
- jobTitle: ["Mortgage Loan Officer", "Realtor"]
- telephone: both phone numbers from CONTACT
- email: both emails from CONTACT
- url: BASE_URL
- image: `${BASE_URL}/headshot.jpg`
- sameAs: array of SOCIAL_LINKS values (Facebook, Instagram, LinkedIn)
- worksFor: array with two Organization objects (Loan Factory and oNest)
- hasCredential: array with two EducationalOccupationalCredential objects (NMLS #2528620 and VA Real Estate License #225273183)

**LocalBusiness schema:**
- @type: "LocalBusiness", @id: `${BASE_URL}/#business`
- name: COMPANIES.loanFactory.name
- telephone: CONTACT.phoneHref
- email: CONTACT.email
- address: PostalAddress with streetAddress "9697 Main St", Fairfax, VA 22031, US
- areaServed: array of 12 state codes ["VA", "DC", "FL", "GA", "KY", "MA", "MD", "NC", "NH", "OH", "PA", "WA"]
- employee: reference to `${BASE_URL}/#person`

**RealEstateAgent schema:**
- @type: "RealEstateAgent", @id: `${BASE_URL}/#realestateagent`
- name: "Prashanna Sangroula"
- telephone: CONTACT.realEstatePhoneHref
- email: CONTACT.realEstateEmail
- worksFor: Organization with COMPANIES.onest.name
- areaServed: { @type: "State", name: "Virginia" }

4. Render the script tag in the Home component JSX (inside the fragment, before HeroSection). Use the official Next.js JSON-LD pattern: a script element with type="application/ld+json" and the stringified JSON passed via the innerHTML mechanism recommended in the Next.js docs. Apply the XSS-safe replacement of angle brackets (`<` replaced with unicode escape `\u003c`) per official docs -- defensive measure even though all data comes from static constants.
